Reverse Polish Notation Calculator

This is a Reverse Polish Notation calculator. It is different than most calculators you're probably use to. This calculator uses the enter key to indicate the end of the first number and the operation keys to tell the computer what operation you want to use on both numbers.

Example

To do the calculation 2+2 in RPN you would press 2 Enter 2 +.

To string multiple calculations togeter such as 2+2+2 you would press 2 enter 2 + enter 2

This is a style of calculator that companies such as HP tried to push however it never caught on
